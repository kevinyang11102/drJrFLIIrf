# 旅游信息管理系统 java1048

## 项目概述

旅游信息管理系统是一款基于前后端分离模式开发的应用，旨在提供一站式的旅游信息管理服务。本项目包含详尽的文档、源码、数据库文件，并提供了完整的调试与运行支持。

## 技术栈

### 后端技术

- SpringBoot
- MyBatis

### 前端技术

- Vue
- Axios
- ElementUI
- JSP

### 开发环境

- Java 1.8
- Node.js
- MySQL
- Maven

## 功能模块

### 前台

- 展示旅游新闻
- 景区信息展示
- 美食信息展示
- 旅游线路展示
- 在线留言功能

### 后台

#### 管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

#### 普通用户

- 旅游线路查询
- 订单信息查询

## 系统角色

- 管理员：负责系统的管理与维护
- 普通用户：使用系统进行旅游信息查询及预订

## 运行环境

- 后端：Java 1.8+、MySQL 5.7+
- 前端：现代浏览器
